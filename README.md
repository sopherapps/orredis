# orredis

Just a a simple fast ORM for python built in rust

## How to demo

```shell
python3 -m venv env 
source env/bin/activate
pip install -r requirements.txt
maturin develop
python
import orredis
# do stuff with orredis
```

## How to Test

```shell
python3 -m venv env 
source env/bin/activate
pip install -r requirements.txt
maturin develop
pytest
```
